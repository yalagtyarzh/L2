```go
package main

func main() {
	ch := make(chan int)
	go func() {
		for i := 0; i < 10; i++ {
			ch <- i
		}
	}()

	for n := range ch {
		println(n)
	}
}
```

Ответ:
```
В выводе мы получим цифры от 0 до 9, а потом словим deadlock, поскольку в основной горутине мы продолжаем читать канал
ch, но горутин, записывающих данные в канал, больше нет. Для того, чтобы корректно прочитать данные, в горутине нужно
после заполнения канала его закрыть
```
